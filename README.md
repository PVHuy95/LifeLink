**LifeLink** is a web platform designed to provide emergency ambulance booking services, connecting users with the nearest medical units quickly and efficiently.

> [!NOTE]
> **This project was developed for the Techwiz 5 Competition.**
> **This is a Frontend Demo Project.**
> This repository contains only the user interface (UI) and frontend logic. It simulates data and interactions for demonstration purposes and does not connect to a real backend server or database.

## ğŸš€ Installation & Setup

Follow these steps to run the project locally:

### 1. Prerequisites
*   Node.js (Recommended: v16.x or higher)
*   npm or yarn

### 2. Installation

Clone the repository:
```bash
git clone https://github.com/PVHuy95/LifeLink.git
cd LifeLink
```

Install dependencies:
```bash
npm install
```

### 3. Running the App

Start the development server:
```bash
npm run dev
```

Open your browser and navigate to: `http://localhost:5173/LifeLink/`
